# 🩺 Diabetes Prediction System using Machine Learning

## 📌 Overview

This project predicts whether a patient is diabetic based on medical parameters using Machine Learning. The project uses the PIMA Indians Diabetes Dataset and compares multiple classification algorithms.

## 🚀 Features

- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Correlation Heatmap
- Histograms and Boxplots
- Feature Scaling
- Multiple ML Model Comparison
- Diabetes Prediction System

## 📊 Dataset

- Dataset: PIMA Indians Diabetes Dataset
- Records: 768
- Features: 8
- Target:
  - 0 → Non-Diabetic
  - 1 → Diabetic

## 🤖 Machine Learning Models

| Model | Accuracy |
|-------|----------|
| Logistic Regression | 75.97% |
| Decision Tree | 69.48% |
| Random Forest | 72.73% |
| Support Vector Machine | **77.27%** |

**Best Model:** Support Vector Machine (SVM)
